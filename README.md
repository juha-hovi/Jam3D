## Directory structure
### bin: Binaries
Folder for cmake build files. Executable output is /bin/src/Jam3D

### docs: Documents
Files describing the software usage or functionality

### res: Resources
Asset files such as texture images.

### src: Source
All self-written project source code.

### vendor: Vendor
3rd party libraries.
